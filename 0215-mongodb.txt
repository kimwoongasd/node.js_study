서버실행
- mongod --dbpath 폴더경로
ex) mongod --dbpath c:/mydb


mongo gui
- use 컬렉션명 : 테이블이 있으면 이동, 없으면 생성 후 이동

insert
- db.컬렉션명.insert({값:값}) : 데이터 추가, insertone 또는 insertmany 사용을 권장
- insert할 경우 컬럼을 추가해서 값을 추가할 수 있다(확장이 용이하다)

find
- db.컬렉션명.find() : 조회
- db.컬렉션명.find({}, {컬럼명:1, 컬럼멸: 0}) : 선택한 컬럼 조회, _id는 자동 조회, 0은 조회 x
* 조건 있을 경우
- db.컬렉션명.find({ 컬럼명: 조건 }, {컬럼명:1, 컬럼멸: 0})
* 조건이 아닌 경우
- db.컬렉션명.find({ 컬럼명: {$ne:조건} }, {컬럼명:1, 컬럼멸: 0})


연습	
1. 가격이 10000원 이상인 모든 도서 조회
db.book.find({price : {$gte:10000}})
2. 출판사가 굿스포치 인 모든 도서 조회
db.book.find({publisher:"굿스포츠"})
3. 가격이 10000원 이상이고 출판사가 굿스포츠인 모든 도서 조회
4. 가격이 10000원 이상이고 출판사가 굿스포츠인 모든 도서를 가격순으로 정렬
5. 도서명에 축구를 포함하는 모든 도서
6. 도서명에 축구를 포함하고 있는 도서의 도서명 가격을 조회
7. 도서명에 축구를 포함하고 도서명 가격이 높은순으로 정렬
8. 굿스포츠나 이상미디어의 모든 도서를 가격이 높은순으로 출력
9. 가격이 10000원에서 20000원 사이 모든 도서
10. 이상미디어에서 출간하는 도서의 수
